    网络通信 #

    I/O 模型 #

    《UNIX 网络编程》中总结了 5 种 I/O 模型，包括同步和异步 I/O：

    • 阻塞 I/O (Blocking I/O)
    • 非阻塞 I/O (Nonblocking I/O)
    • I/O 多路复用 (I/O multiplexing)
    • 信号驱动 I/O (Signal driven I/O)
    • 异步 I/O (Asynchronous I/O)

    操作系统上的 I/O 是用户空间和内核空间的数据交互。
