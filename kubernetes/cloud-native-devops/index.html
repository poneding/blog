    7. Kubernetes 运维最佳实践 #

    7.1 资源优化 #

    • 合理设置 Requests & Limits
    • 使用 HPA/VPA 进行自动扩展
    • 合理选择 Pod 亲和性和调度策略

    7.2 安全管理 #

    • 最小权限 RBAC 配置
    • 启用 NetworkPolicy 进行网络隔离
    • 避免使用 root 用户运行容器

    7.3 日志与监控 #

    • EFK/Loki 进行日志采集
    • Prometheus + Grafana 进行监控
    • OpenTelemetry 进行链路追踪

    7.4 备份与恢复 #

    • 定期备份 etcd
    • 使用 Velero 备份 PVC 数据
    • 确保 CI/CD 变更可回滚

    7.5 高可用 #

    • 多 Master 节点 + Keepalived + HAProxy
    • 使用 StatefulSet 管理有状态应用
    • 存储选型（Ceph, NFS, Longhorn）
