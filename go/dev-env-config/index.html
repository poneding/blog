    Go 开发环境配置 #

    cobra-cli #

    安装：

    go install github.com/spf13/cobra-cli@latest

    自动补全：

    cobra-cli completion zsh > .zfunc/_cobra-cli

    在 .zshrc 文件中添加内容（如果已添加，则忽略）：

    fpath+=~/.zfunc
autoload -Uz compinit && compinit
